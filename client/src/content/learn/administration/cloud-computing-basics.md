# Cloud Computing Basics

Welcome to Lesson 1 of the Administration course.

## Introduction to Cloud Computing

Cloud computing means using the internet to access computers, storage, and software instead of buying your own hardware.

Think of it like electricity — you do not build your own power plant. You simply plug in and use the power you need.

> **Example:** Instead of buying a server to run your website, you rent one from a cloud provider and pay only for what you use.

## What is Cloud Computing

Cloud computing lets you use computing resources — like servers, storage, and software — over the internet.

When you save a photo to Google Drive or stream a movie on Netflix, you are using cloud computing. The files are not stored on your device — they live on powerful computers owned by cloud providers.

Common examples you already use: Gmail, Google Drive, Netflix, and Dropbox.

## Private and Public Cloud

There are two main types of cloud:

**Private Cloud** — Used by one organization only. It is more secure and gives more control. Best for banks, hospitals, or any place that handles sensitive data.

**Public Cloud** — Managed by a provider like AWS or Google. Anyone can use it over the internet. It is cheaper and easier to set up.

| Feature  | Private Cloud        | Public Cloud          |
|----------|----------------------|-----------------------|
| Cost     | Higher               | Lower                 |
| Security | Higher               | Good                  |
| Best For | Sensitive data       | Websites, apps        |

## What is the Difference Between SAAS and PAAS

**SaaS (Software as a Service)** — Ready-made software you use directly. No installation needed. Examples: Google Docs, Zoom, Microsoft 365.

**PaaS (Platform as a Service)** — A platform for developers to build and deploy their own apps. Examples: Heroku, Google App Engine.

| | SaaS | PaaS |
|---|---|---|
| Who uses it | Everyone | Developers |
| Coding needed | No | Yes |
| Example | Gmail | Heroku |

## What are the Benefits of Cloud Computing?

- **Save Money** — No need to buy expensive hardware.
- **Scale Easily** — Add or remove resources as needed.
- **Access Anywhere** — Work from any device with internet.
- **Reliable** — Cloud providers keep backups so your data stays safe.
- **Fast to Start** — Launch apps in minutes, not weeks.
